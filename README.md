# Nasa Hackathon Project

## Run the app locally

### Requirements

```
Node.js > v.18
Npm > v.10
```

### Run those commands in order

```bash
cp .env.example -r .env.local
# Update the .env.local file accordingly
```

```
npm install -g pnpm
pnpm install
pnpm dev
```

### Preview the app

Open [http://localhost:3000](http://localhost:3000) in your browser.
